Partools:

a Sensible Package for Large Data Sets

Role: co-author

The partools package is a general framework for parallel processing of large data sets and/or highlycomputational algorithms. It is the "un-MapReduce," avoiding that paradigm while retaining the philosophy of highly-distributed memory objects and files of Hadoop and Spark.

DayLight:

3D Object Editor & Batch Renderer

Role: Author

DayLight is built with an abstraction of OpenGL functions. It manages OpenGL resources so the users do not have to manually free openGL resources. In addition, it hides the complicated construction for objects like framebuffer objects, texture buffer objects such that the users can use OpenGL resources with simple functions like use(), stopUsing(), or isInUse().

DayLight utilizes X11 and EGL for window management. It supports batch rendering of large scene with great efficiency in headless modes

DayLight Utilizes deferred shading to improve real time rendering of multiple light sources and post processing (shadow mapping, bloom effect etc). It supports basic edits (translation, rotation, scaling, duplication) on instantiated objects to minimize GPU memory usage.

Note: Documentation will be released around May 2018

xSQL:

an imitation of MySQL because I can

Role: Author

xSQL is built with a parser that imitates the behavior of a real database server using Bison (C++ version of YACC) and Flex (C++ version of LEX) to generate an optimizable and compilable expression tree

It is nicely integrated with functionalities of a database, such as creation, projection, selection, and indexing on selected columns Processed the expression tree with a set of rewrite rules before its execution and optimizes it by using a hash or B+ Tree index when available

Machine Learning Toy projects:

Collections of Machine Learning projects

Symbol Recognizer

Role: Author

The backbone of this program is an 5-layer convolutional neural network in Tensor Flow. It is trained with a large collection of handwritten digits for less 30 minutes but with accuracy of 99%+.

It incoporates a user interface using OpenCV so user can write with mouse.

Gender Classifier

Role: Author

The backbone of this program is an 3-layer convolutional neural network built with Java. It is trained with a large collection of frontal headshot for around 10 minutes but with accuracy of 98%+.

Content Aware Resizing

Role: Implementor

This program is an implementation of Seam Carving for Content-Aware Image Resizing.

It finds and deletes the least resistent seam horizontally or vertically such that the alteration of the image is least noticeable.
